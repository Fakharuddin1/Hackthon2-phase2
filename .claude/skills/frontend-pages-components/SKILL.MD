# Frontend Pages & Components Skill

## Instructions

1. **Page structure**
   - Semantic HTML structure
   - Clear sections (header, main, footer)
   - SEO-friendly markup

2. **Component design**
   - Reusable UI components
   - Props-based or modular structure
   - Consistent naming conventions

3. **Layout system**
   - Flexbox and Grid usage
   - Responsive breakpoints
   - Mobile-first layout approach

4. **Styling**
   - Clean and scalable CSS
   - Utility-first or component-based styles
   - Consistent spacing, colors, and typography

## Best Practices
- Use semantic HTML elements
- Keep components small and reusable
- Follow mobile-first design
- Maintain consistent design tokens (colors, fonts)
- Avoid inline styling for large projects
